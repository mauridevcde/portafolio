Mis Proyectos

FrontEnd Sistema de Gestión Veterinaria

Frontend desarrollado con Next.js y React, con una interfaz moderna y responsiva. Se utilizan Next Middleware para proteger rutas, Zustand para el manejo del estado global, y TanStack Query junto con Axios para la gestión eficiente de llamadas a la API. La aplicación está conectada de forma segura al backend usando tokens de autenticación para ofrecer una experiencia fluida y protegida al usuario.

BackEnd Sistema de Gestión Veterinaria

Backend desarrollado con Node.js y Express, utilizando TypeScript para una estructura tipada y mantenible. Las rutas están protegidas mediante tokens de autenticación, y las validaciones de datos se realizan con Zod. El proyecto usa MySQL2 con un modelo relacional que conecta entidades como mascotas, veterinarios, productos y consultas. También se implementaron middlewares personalizados para la validación y el control de acceso.
