## Introducción a Flutter y Dart

<p>Flutter es un framework de código abierto desarrollado por Google que te permite crear aplicaciones móviles para Android e iOS utilizando un solo código base. <strong>Dart</strong> es el lenguaje de programación utilizado por Flutter para desarrollar aplicaciones móviles.</p> <ul> <li>Flutter es multiplataforma</li> <li>Dart es fácil de aprender</li> <li>Flutter y Dart ofrecen una gran comunidad de desarrolladores</li> </ul>

## Instalación y configuración de Flutter y Dart

<p>Para empezar a desarrollar con Flutter y Dart, debes instalar el SDK de Flutter y configurar tu entorno de desarrollo. Puedes hacerlo siguiendo los siguientes pasos:</p> <pre><code>git clone https://github.com/flutter/flutter.git</code></pre> <p>Luego, debes agregar el path del SDK de Flutter a tu sistema operativo.</p>
